# Personal CV Website

## Overview

This project is a **personal CV / resume website** built using **HTML and CSS**. It presents my professional profile in a clean, simple, and readable format. The website includes sections such as Profile, Education, Technical Projects, Skills & Expertise, and Languages.

If this project helped you or inspired you in any way, please consider giving this repository a ⭐. It motivates me to continue learning and building more projects.
